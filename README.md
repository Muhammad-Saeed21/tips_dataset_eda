# Tips Dataset EDA (Exploratory Data Analysis)

This project performs **Exploratory Data Analysis (EDA)** on the built-in **Seaborn Tips dataset**, which contains restaurant billing, tipping, and customer behavior data.  
The goal is to understand basic patterns using summary statistics and simple visualizations.
